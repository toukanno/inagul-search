# ⚡ イナグル — イナズマイレブン キャラクター検索

イナズマイレブンシリーズのキャラクターを検索・フィルター・ソートできるサイトです。

## 機能

- **テキスト検索** — 名前・チーム名・必殺技で絞り込み（クリアボタン付き）
- **ポジションフィルター** — GK / DF / MF / FW で絞り込み
- **属性フィルター** — 火 / 土 / 木 / 風 / 闇 で絞り込み
- **シリーズフィルター** — イナイレ1 / GO / クロノ・ストーン / ギャラクシー で絞り込み
- **ソート** — 登場順 / 名前（五十音順）/ チーム名 / 属性 / 背番号 / シリーズ
- **昇順・降順トグル** — すべてのソートで昇順/降順を切り替え可能
- **アクティブフィルター表示** — 現在の絞り込み条件を表示、一括リセット対応
